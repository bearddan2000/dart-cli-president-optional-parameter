# dart-cli-president-optional-parameter

## Description
Optional parameter is allowed.
- must be at the end of arguments
- must provide default value
- must be in `{}`

## Tech stack
- dart

## Docker stack
- google/dart

## To run
`sudo ./install.sh -u`

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
